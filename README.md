# Body Fat Predictor

A machine learning project that predicts body fat percentage from body measurements using regression techniques.

## Overview
This project uses a dataset from Kaggle to estimate body fat percentage based on physical measurements such as age, weight, height, and other body circumferences.

## Dataset
- **Source:** [Body Fat Prediction Dataset - Kaggle](https://www.kaggle.com/datasets/fedesoriano/body-fat-prediction-dataset)
- The dataset contains body measurements and corresponding body fat percentages for 252 men.

## Contents
- `bodyfat_prediction.ipynb` — Main notebook containing:
  - Exploratory Data Analysis (EDA)
  - Data preprocessing
  - Model training and evaluation
  - Results and conclusions

## Technologies Used
- Python
- Jupyter Notebook
- Pandas & NumPy
- Scikit-learn
- Matplotlib & Seaborn

## How to Run
1. Clone the repository:
```bash
   git clone https://github.com/AungKoMin200/Body-Fat-Predictor.git
```
2. Download the dataset from [Kaggle](https://www.kaggle.com/datasets/fedesoriano/body-fat-prediction-dataset) and place it in the project folder
3. Open `bodyfat_prediction.ipynb` in Jupyter Notebook or Google Colab
4. Run all cells
